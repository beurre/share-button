Guides lead a user through a specific task they want to accomplish, often with a sequence of steps.
Writing a good guide requires thinking about what your users are trying to do.

## Install from NPM

Recommended for ease of maintainability.

<Steps>

1. Install the package from NPM

    ```bash
    npm install @royalfig/share-button
    ```

2. Import the package into your project

    ```js
    import "@royalfig/share-button"
    ```

3. Add the custom element to the page

    ```html
    <share-button></share-button>
    ```

</Steps>

## Install from release

Recommended if you have less experience with coding and can add scripts and HTML to your site.

<Steps>

1. Go to the release page. View or download the JS file.

</Steps>
