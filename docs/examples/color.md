# Color customization

## Change button color

```html
<share-button color-button="red"></share-button>
```

<div class="sb-container">
    <share-button color-button="red"></share-button>
</div>

## Change pop up color

```html
<share-button color-surface="blue" color-element="white"></share-button>
```

<div class="sb-container">
    <share-button color-surface="blue" color-element="white"></share-button>
</div>
